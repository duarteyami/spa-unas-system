# Sistema de Gestión - Spa de Uñas

Sistema web desarrollado para la gestión de un spa de uñas, 
como proyecto universitario de la materia Taller de Programación.
Actualmente en proceso de mejoras y actualizaciones.

## Tecnologías utilizadas
- Java / JSP
- PostgreSQL
- GlassFish Server
- NetBeans IDE
- Bootstrap
- HTML / CSS / JavaScript

## Funcionalidades
- Gestión de clientes
- Gestión de empleados
- Órdenes de servicio
- Facturación
- Reportes

## Estado del proyecto
En desarrollo - se continúan agregando mejoras.
